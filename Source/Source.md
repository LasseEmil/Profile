# Source Directory

- `WebSite/`: shared assets for the WASM app shell (global styles + generated `updates.html`).
- `Updates/`: Markdown log + converter script for personal updates consumed by the site.
- `WasmSite/`: C++ + template files that compile to the WebAssembly experience rendered at runtime.

Follow the naming conventions in `AGENTS.md` (PascalCase directories, snake_case scripts). Any new subdirectory-specific docs should follow the same pattern (`DirName/DirName.md`).
