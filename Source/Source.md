# Source Directory

- `WebSite/`: main static HTML/CSS/JS for the profile (mirrors the GitHub Pages output).
- `Updates/`: Markdown log + converter script for personal updates consumed by the site.
- `WasmSite/`: C++ + template files compiled to WebAssembly for the companion experience.

Follow the naming conventions in `AGENTS.md` (PascalCase directories, snake_case scripts). Any new subdirectory-specific docs should follow the same pattern (e.g., `WebSite/WebSite.md`).
